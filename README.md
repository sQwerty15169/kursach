# Курсовой проект: Менеджер заметок (Python + tkinter + Docker)

## Что реализовано
Рабочее настольное приложение на Python с графическим интерфейсом:
- создание, редактирование и удаление заметок;
- хранение данных в `notes.json`;
- поиск по заголовку и содержимому;
- переключение тем оформления (light/dark);
- логирование действий в `app.log`.

## Соответствие заданию
- **Работоспособность:** приложение запускается обычной командой `python main.py`.
- **Итеративность:** добавлены минимум две расширенные функции:
  - поиск по заметкам;
  - смена темы;
  - логирование (дополнительно).
- **Документированность:** есть комментарии в коде и файл `AGENTS.md`.
- **Контейнеризация:** добавлен `Dockerfile` для запуска в контейнере.

## Локальный запуск
```bash
python main.py
```

## Запуск через Docker
Сборка образа:
```bash
docker build -t kp-notes-app .
```

Запуск контейнера (для Linux/X11 хоста):
```bash
docker run --rm -e DISPLAY=$DISPLAY -v /tmp/.X11-unix:/tmp/.X11-unix kp-notes-app
```

> Примечание: для GUI-приложений в Docker на Windows/macOS требуется отдельная настройка X-сервера.

## Файлы проекта
- `main.py` — приложение.
- `Dockerfile` — сборка контейнера.
- `AGENTS.md` — описание структуры для ИИ-агентов.
- `Пояснительная_записка.md` — отчетные материалы.
