# AGENTS.md

## Контекст проекта
Проект: настольное Python-приложение "Менеджер заметок" для курсовой работы по дисциплине "Современные технологии программирования".

## Структура
- `main.py` — основной исполняемый модуль с GUI на `tkinter`.
- `notes.json` — база заметок в формате JSON (создается автоматически).
- `config.json` — конфигурация интерфейса, хранит выбранную тему (создается автоматически).
- `app.log` — файл логирования действий пользователя (создается автоматически).
- `Dockerfile` — контейнеризация и автоматический запуск приложения.
- `README.md` — описание проекта, запуск и карта требований.
- `Пояснительная_записка.md` — отчетная часть по этапам задания.

## Логика модулей
- `Note` (dataclass): модель заметки.
- `NotesRepository`: чтение/сохранение заметок.
- `ConfigRepository`: хранение пользовательских настроек.
- `NotesApp`: GUI, CRUD-операции, поиск, переключение темы.

## Основные функции
1. Создание, редактирование и удаление заметок.
2. Поиск по заголовку и тексту.
3. Сохранение данных между запусками в JSON.
4. Смена светлой/темной темы.
5. Логирование действий в `app.log`.

## Правила развития
- Новые функции добавлять через отдельные методы `NotesApp`.
- Формат данных заметки поддерживать обратно совместимым.
- Логи должны оставаться человекочитаемыми.
